/* ************************************************************ *
 *                    rainbow3                                  *
 *  README                                                      *
 *  desc: for attiny45 (8 Mhz), NeoPixel connected on pin 6     *
 *        run a rainbow of colors until detection of a pressure *
 *        on the push button (pin 5) that stops the rainbow on  *
 *        the current color.                                    *
 *        Another pressure detection will start over the rainbow*
 *                                                              *
 * pinout:                                                      *
 *               ______                                         *
 *        n/a --|1    8|-- Vcc                                  *
 *        n/a --|2    7|-- Debug LED                            *
 *        n/a --|3    6|-- NeoPixel Din                         *
 *        gnd --|4____5|-- push button                          *
 *                                                              *
 * ************************************************************ */
